==============================================================================
INTRODUCTION                                                        *shavian*

The Shavian input plugin allows you to type in Shavian script using a QWERTY
keyboard layout. The plugin provides a command to toggle Shavian input mode
on or off. It works exclusively in insert mode.

For more information about the QWERTY layout used, visit:
    https://shavian.info

==============================================================================
COMMANDS                                                            *shavian-commands*

                                                        *:ToggleShavianMode*
:ToggleShavianMode
    Toggles Shavian input mode on or off. When enabled, typing in insert mode
    will map QWERTY keys to their corresponding Shavian characters.
